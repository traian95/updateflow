Asset-uri pentru build (logo si icon)
=====================================

Poti pune aici fisierele necesare pentru exe si instalator, daca nu le ai in radacina proiectului:

  logo.ico     - Icon pentru aplicatie si pentru wizard instalare (OBLIGATORIU)
  Naturen2.png - Logo afisat in fereastra de login si in aplicatie (OBLIGATORIU)
  despre.gif   - Animatia pentru ecranul "Despre" din Ofertare (opțional)

Daca le pui in radacina proiectului ("Soft Ofertare Usi"), nu e nevoie de acest folder.

Pentru logo.ico: daca ai doar Naturen2.png, poti converti PNG -> ICO cu:
  - un editor (GIMP, Paint .NET) sau
  - un site online (cauta "png to ico converter").
